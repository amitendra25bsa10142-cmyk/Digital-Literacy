📖 Project Overview

This repository contains my Digital Literacy Project submitted as part of the CSE0001 – Digital Literacy course at VIT Bhopal University.

As a Student Digital Ambassador, this project demonstrates my ability to:
	•	Create digital content
	•	Build professional online presence
	•	Use coding & collaboration platforms
	•	Communicate professionally
	•	Understand cyber safety and digital responsibility

This project includes five tasks, each aligned with one module of the Digital Literacy course.

Task 1 – Digital Literacy Awareness Infographic

In this task, I created a Digital Literacy Awareness Infographic using Canva.
The infographic explains:
	•	What is Digital Literacy
	•	Safe Internet Practices
	•	Professional Online Presence
	•	Useful Digital Tools

This visual resource is designed to help students understand the importance of digital literacy.


Task 2 – Student Digital Portfolio

In this task, I created and updated my professional profiles:
	•	GitHub Profile
	•	LinkedIn Profile
	•	Kaggle Profile

These platforms help build a professional online presence and prepare for internships and career opportunities.
